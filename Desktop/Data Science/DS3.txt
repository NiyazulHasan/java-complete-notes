Road map for Data Science 
=========================
1. Foundations
	
	> Math (important, but not PhD level):

		* Statistics (mean, median, variance)
		* Probability
		* Linear Algebra (vectors, matrices)

2. Programming Language

	> Learn Python
		* Variables, loops, functions
		* File handling
		* Basic OOP

3. Data Handling & Analysis

	This is where real Data Science starts.

	> Libraries:
		* NumPy
		* Pandas
	> Learn:
		* Data cleaning
		* Handling missing values
		* Filtering, grouping data


3. Data Visualization

	> Tools:
		* Matplotlib
		* Seaborn
	> Learn:
		* Charts (bar, line, histogram)
		* Storytelling with data


4. Machine Learning (Core Part)

	> Library:
		* Scikit-learn
	> Concepts:
		* Supervised learning (Regression, Classification)
		* Unsupervised learning (Clustering)
		* Model evaluation

5. Advanced Topics 

	> Deep Learning:
		* TensorFlow
		* PyTorch
	> NLP (Chatbots, text analysis)
	> Computer Vision


6. Databases & SQL

	> Learn:
		* SQL (SELECT, JOIN, GROUP BY)
		* Basics of databases like Oracle or MySQL



7. Projects (Most Important)

	Without projects, skills are incomplete.

	> Build:
		* Sales prediction model
		* Student performance analysis
		* Movie recommendation system


8. Tools & Platforms

	> Jupyter Notebook
	> GitHub (for portfolio)
	> Excel (yes, still important!)



Interview Question 
===================

Q) What is Data Science?

Data Science is the process of collecting, analyzing, and interpreting data to extract useful insights and support decision-making.

or

Data Science helps in making better decisions, predicting future outcomes, and solving real-world problems using data.
