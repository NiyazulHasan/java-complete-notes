Types of programming in python
===============================

Python supports multiple programming paradigms.

1. Procedural Programming
--------------------------
Program is written step-by-step using functions and statements.

Example:

	def add(a, b):
    		return a + b
	print(add(2, 3))

Features
--------
> Simple
> Easy for beginners
> Focus on functions and procedures


2. Object-Oriented Programming (OOPS)
--------------------------------------
Program is organized using classes and objects.

Example
	class Student:
    		def __init__(self, name):
        		self.name = name

		s1 = Student("Rahul")
		print(s1.name)
Features
> Reusable code
> Real-world modeling
> Better organization



3. Functional Programming
--------------------------
Programming using functions as main building blocks.
Example
	numbers = [1, 2, 3, 4]
	square = list(map(lambda x: x*x, numbers))
	print(square)

Features
> Uses lambda, map, filter, reduce
> Less mutable data
> Cleaner code for some tasks



4. Scripting Style
-------------------
Python is widely used as a scripting language for automation.
Example
	print("Backup completed")
Uses
> Automation
> File handling
> System administration


5. Modular Programming
-----------------------
Program is divided into modules/files.

Example
	# math_utils.py
	def add(a, b):
   	 return a + b

	# main.py
	from math_utils import add
	print(add(2, 5))
Features
> Better code management
> Reusability
