Kruskal–Wallis (H-test)
=======================
The Kruskal–Wallis test is a non-parametric statistical test used to compare three or more independent groups when the assumptions of a one-way ANOVA (especially normality) are not met.
	
	12	 R²_1	R²_2	R²_3		R²_k
H = -------- x (----- + ---- + ------ + ..... + --- 	) - 3 (N + 1) 
      N(N+1)      n_1   n_2      n_3             n_k

	n1,n2,n3,....,nk = Number of each of k samples
	N = n1 + n2 + n3 + ...... + nk
	R1, R2, R3, ....., Rk = Rank sums of each sample

For small samples (not less than 5). H is approximately distributed as Chi-Square with "K-1" degree of freedom. (V = k -1).
