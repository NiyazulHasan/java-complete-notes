Classification of Hypothesis Testing 
====================================
Hypothesis testing is broadly classified into two main types.

1) Parametric Test  (To find out age of a person we use years -  parametric test)

2) Non-Parametric Test (To find out human beauty we don't have parameters - non-parametric Test)


1) Parametric Test
-------------------
Parametric tests are statistical tests that assume the population follows a specific distribution.

Characteristics (Features)
--------------------------
These tests are based on parameters such as mean and standard deviation.
They require numerical or quantitative data.
If the assumptions are correct, these tests provide more accurate and powerful results.

Note:
-----
	Quantitative data means numerical data — data that can be measured or counted using numbers.
	ex:
		Age = 20 years
		Height = 170 cm
		Weight = 65 kg
		Marks = 85
		Salary = ₹50,000

Assumptions (Conditions)
-------------------------
The data should be normally distributed.
The samples should be independent of each other.
The variance of all groups should be almost the same (homogeneous variance).

Note:
----
	Normal distribution is a type of data distribution in which most data 
	values are concentrated around the center, and the graph looks like a bell shape.
	It is also called a bell curve.
	ex:
		If you take the marks of students in a class:

		Most students score average marks.
		Very few students score very high or very low marks.
		So, the graph forms a bell shape — this is called a normal distribution.	

Examples
		Z-test (>=30 large sample size)
		t-test (<30 small sample size)
		F-test (Comparing variance between s1 and s2)	
		ANOVA  (3+ group means compare)
		Pearson correlation (relationship between two variables)



2) Non-Parametric Tests
-------------------------
Non-parametric tests do not require assumptions about the population distribution.

Characteristics
------------------
Suitable for ordinal or nominal data
Can be used with small sample sizes
Useful when data are not normally distributed

Note:
----
	Nominal data → categories without order
		Example: gender, blood group, colors
	Ordinal data → categories with order/ranking
		Example: poor, average, good OR class ranks

Examples

	Chi-square test
	Mann–Whitney U test
	Wilcoxon signed-rank test
	Kruskal–Wallis test
	Spearman rank correlation
