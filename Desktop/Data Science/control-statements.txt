Control statements
=================
Control statement enables the programmer to control flow of a program.

Control statement allows us to make decisions, to jump from one section of code to another section and to execute the code repeatedly.

We have three control statements in python.

1) Conditional Statements	
2) Looping statements 
3) Jump statements

1. Conditional Statements (Decision Making)
--------------------------------------------
Conditional statements execute code based on conditions.

i. if Statement
-------------
Executes code only if condition is True.

Syntax
if condition:
    statement
Example
age = 18

if age >= 18:
    print("You can vote")


ii. if-else Statement
---------------------
Executes code either our condition is True or False.

Syntax
if condition:
    statements
else:
    statements
Example
number = 5

if number % 2 == 0:
    print("Even")
else:
    print("Odd")



iii. if-elif-else Ladder
----------------------
Used for multiple conditions.

Syntax
if condition1:
    statements

elif condition2:
    statements

elif condition3:
    statements

else:
    statements
Example
marks = 75

if marks >= 90:
    print("Grade A")

elif marks >= 70:
    print("Grade B")

elif marks >= 50:
    print("Grade C")

else:
    print("Fail")


iv. Nested if
------------
An if inside another if.

Example
age = 20
citizen = True

if age >= 18:
    if citizen:
        print("Eligible to vote")





2. Looping Statements
-------------------------
Loops repeat code multiple times.

Python has:

	for loop
	while loop`

i. for Loop
--------------
Used when number of repetitions is known.

Syntax
for variable in sequence:
    statements

Example 1 — Printing Numbers
for i in range(5):
    print(i)

ex:

for i in range(1, 10, 2):
    print(i)

ex:
---
name = "Python"

for ch in name:
    print(ch)


ex:
----
fruits = ["apple", "banana", "mango"]

for fruit in fruits:
    print(fruit)




ii. while Loop
----------------
Runs while condition is true.

Syntax
while condition:
    statements


ex:
---
count = 1

while count <= 5:
    print(count)
    count += 1

Infinite Loop
------------
If condition never becomes false.

while True:
    print("Hello")




3. Jump Statements
-------------------
Jump statements alter loop execution.

Python provides:

	break
	continue
	pass

i. break Statement
-----------------
Stops the loop completely.

Example
for i in range(10):

    if i == 5:
        break

    print(i)


ii. continue Statement
-----------------------
Skips current iteration.

Example
for i in range(5):

    if i == 2:
        continue

    print(i)



iii. pass Statement
----------------
Does nothing.

Used as placeholder.

Example
for i in range(5):

    if i == 3:
        pass

    print(i)



Loop Control with else
------------------------
Python loops can have else.

The else block runs if loop ends normally.

Example
for i in range(5):
    print(i)

else:
    print("Loop completed")


Example with break
for i in range(5):

    if i == 3:
        break

else:
    print("Completed")
