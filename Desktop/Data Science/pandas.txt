Pandas 
======
The name comes from Panel Data (multi-dimensional data), Not related to the animal.

Pandas is a powerful open-source library in Python used for data analysis and manipulation. It’s one of the most important tools in the field of Data Science and is widely used for handling structured data like tables, spreadsheets, and time series.


Pandas provides easy-to-use data structures and functions that allow you to:

> Read data from different sources (CSV, Excel, databases, etc.)
> Clean and preprocess messy data
> Analyze and manipulate datasets
> Perform statistical operations
> Visualize data (often with libraries like Matplotlib)



Advantages of Pandas
----------------------
> Very fast and efficient for large datasets
> Easy syntax (built on top of Python)
> Integrates well with libraries like NumPy and Seaborn.


Real-Life Use Cases
-------------------
Pandas is used in:

> Data analysis and reporting
> Machine learning preprocessing
> Financial analysis
> Data cleaning in research
> Business analytics

Pandas Aliasing
---------------
Pandas aliasing usually refers to the common practice of giving the pandas library a short name (pd) when importing it.
ex:
	import pandas as pd



Pandas mainly provides two key data structures:

1. Series (1D)
2. DataFrame (2D)

1. Series (1D)
--------------
A Series is like a single column of data.
Ex:	
	import pandas as pd

	data = pd.Series([10, 20, 30, 40])
	print(data)

Output:
	0    10
	1    20
	2    30
	3    40


To create an empty Series object in pandas, you can use the Series() constructor.
ex:

	import pandas as pd
	empty_series = pd.Series()
	print(empty_series)

or 

	import pandas as pd
	empty_series = pd.Series(dtype='float64')
	print(empty_series)


Creating Empty Series with Different Data Types
------------------------------------------------

# Integer type
s1 = pd.Series(dtype='int')

# String type
s2 = pd.Series(dtype='str')

# Object type (mixed data)
s3 = pd.Series(dtype='object')

> Adding data later

ex:
	import pandas as pd
	s = pd.Series(dtype='int')
	s.loc[0] = 10
	s.loc[1] = 20
	print(s)

ex:
	s3 = pd.Series(dtype='object')

	s3 = pd.Series(["Apple", "Banana", "Cherry"])
	print(s3)

	> This replaces the empty Series with a filled one.

Using List 
-----------
import pandas as pd

data = ["Apple", "Banana", "Cherry"]
s = pd.Series(data)

print(s)


ex:

s3 = pd.Series(dtype='object')
data = ["Apple", "Banana", "Cherry"]

for i in range(len(data)):
    s3.loc[i] = data[i]

print(s3)


ex:

import pandas as pd
s3 = pd.Series(dtype='object')
data = ["Apple", "Banana", "Cherry"]
new_series = pd.Series(data)
s3 = pd.concat([s3, new_series])

ex:

> Insert list at a specific index range
ex:
import pandas as pd
s3 = pd.Series(dtype='object')
s3 = pd.Series(dtype='object')
data = ["Apple", "Banana", "Cherry"]
s3.loc[0:2] = data
print(s3)

ex:

>Insert list as a single element (important concept)
import pandas as pd
s3 = pd.Series(dtype='object')
s3.loc[0] = ["Apple", "Banana", "Cherry"]
print(s3)


Using dictionary (with custom index)
----------------------------
ex:
	import pandas as pd
	s3 = pd.Series(dtype='object')
	data = {0: "Apple", 1: "Banana", 2: "Cherry"}
	s3 = pd.Series(data)
	print(s3)


We can append the data 
-----------------------
import pandas as pd
s3 = pd.Series(dtype='object')
new_data = pd.Series(["Apple", "Banana"])
s3 = pd.concat([s3, new_data])
print(s3)


Insert at specific index (non-sequential)
---------------------------------------
s3 = pd.Series(dtype='object')
s3.loc[5] = "Mango"
s3.loc[2] = "Orange"
print(s3)



2. DataFrame (2D)
----------------
A DataFrame is like a table (rows and columns), similar to Excel.
Ex:
	data = {
    		"Name": ["Alice", "Bob", "Charlie"],
    		"Age": [25, 30, 35]
		}

		df = pd.DataFrame(data)
		print(df)

output:
  	Name  	Age
0    	Alice   25
1      	Bob   	30
2  	Charlie 35


Key Features of Pandas
----------------------

1. Data Cleaning
------------------
Handle missing data easily:
ex:
	df.dropna()      # remove missing values
	df.fillna(0)     # replace missing values


2. Data Selection & Filtering
------------------------------
Handle rows and columns.
ex:
	df["Age"]              # select column
	df[df["Age"] > 25]     # filter rows

3. Data Manipulation
---------------------
It handle data manipulation and sorting.
ex:
	df["Age"] + 1          # increase age
	df.sort_values("Age")  # sort data

4. Grouping & Aggregation
------------------------
ex:
	df.groupby("Age").count()


5. Reading & Writing Data
------------------------
It performs input/output operations in a file.
ex:
	df = pd.read_csv("file.csv")
	df.to_csv("output.csv")
Note:
	Supports formats like:
	Ex:
		CSV
		Excel
		JSON
		SQL databases



Example1:
---------
import pandas as pd

df = pd.read_csv("students.csv")

# Show first 5 rows
print(df.head())

# Average marks
print(df["Marks"].mean())

# Students with marks > 80
print(df[df["Marks"] > 80])
