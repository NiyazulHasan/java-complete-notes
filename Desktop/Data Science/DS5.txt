Steps to Installation and develop first Python program 
=======================================================
step1:
-----
	Download python software.
	ex:
		https://www.python.org/

step2:
------
	Install python software.
	ex:
		Click to python exe file --> 
		select checkbox-> Use admin privileges when installing py.exe 
		select checkbox-> Add python.exe to PATH  
		--> click on customize install -->  Next --> 
		selec checkbox -> Install python 3.x for all users --> Install --> close.


step3:
------
	Check the setup done perfectly or not.
	ex:
		cmd> python --version 


step4:
-------
	Launch IDLE to write the code

	ex:
		print("Hello World"); 

step5:
------
	We can save the program i.e Demo.py.



Steps to setup Jupyter Notebook
=================================
step1:
------
	Download Anaconda software.
	ex:
		https://www.anaconda.com/download/success?reg=skipped

step2:
-----
	Install Anaconda software.
	ex:
		Click on Anaconda software -->  Next --> I Agree --> Install for (All Users)
		--> Next --> Next --> select checkbox --> clear the package cache upon completion
		--> Install --> Next --> Next --> Uncheck all the checkboxes --> Finish.

		Note:
		-----
			It Launch Anaconda Navigator and Anaconda Distribution.

step3:
------
	Launch Anaconda Prompt.
	(It is CLI that allows us to manage environments and packages within anaconda)
	
step4:
-------
	Open Jupyter notebook. 
	(It is an open source web application to create and share documents containing 
	live code , text and others)
	ex:
		> jupyther notebook

step5:
------
	We can see jupyter notebook by using below url.
	ex:
		http://localhost:8888/tree

step6:
-----
	Create a New Folder on Desktop by clicking New button i.e Python-Anaconda.	
step7:
------
	Select python 3 by clicking New button inside Python-Anaconda folder.

step8:
------
	Rename the file name with Demo.ipynb and double click on it.

step9:
------
	Write the code and press CTRL + Enter to see the output.
