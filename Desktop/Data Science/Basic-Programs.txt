Q)
There is one running race between different participants. So in that race two partners need to finish the race. There is four rounds of race in that each lane length is 100m.So in that one participant will cover 30% of the lane length and another partner will cover remaining length of lane. So print each participant covers distance in that race.


lane_length = 100
rounds = 4

total_distance = lane_length * rounds

participant1 = total_distance * 30 / 100
participant2 = total_distance - participant1

print("Participant 1 covers", participant1, "meters")
print("Participant 2 covers", participant2, "meters")



Q)

Alice is planning to organize a contest with 4 players in each team. 

There are two types of players named Experienced and Freshers. 
To make the contest unbiased Alice wants to have a team in such a way that each team must 
contain at least 1 Experienced and 1 Fresher.
You are given N the number of Experienced and M the number of Freshers. 
Your task is to determine the maximum number of team formations possible.
Example:

Input:
	5 5
Output:
	2

Input:
	10  1

Output: 
	1


E = int(input("Enter number of Experienced players: "))
F = int(input("Enter number of Freshers: "))
teams = min((E + F) // 4, E, F)      
print("Maximum teams =", teams)

Note
----
	Here  // means floor division.

	ex:
		7 // 2 = 3
		Even though 3.5 exists, it goes down to 3.

Q) Write a Python program to accept date of birth of a person and find out his age?


from datetime import date

year = int(input("Enter birth year: "))
month = int(input("Enter birth month: "))
day = int(input("Enter birth day: "))

birth_date = date(year, month, day)
today = date.today()

age = today.year - birth_date.year

print("Age =", age)

Q) Write a Python program to convert seconds to time?

input:
	37823
output:
	10:30:23



total_seconds = int(input("Enter total seconds: "))

hours = total_seconds // 3600
remaining = total_seconds % 3600

minutes = remaining // 60
seconds = remaining % 60

print(f"{hours}:{minutes}:{seconds}")

Q) Sunita wakes up at 6:30 AM and reaches school at 7:15 AM.
Write a Python program to calculate total travel time in minutes.



start_hour = 6
start_minute = 30

end_hour = 7
end_minute = 15

start_total = start_hour * 60 + start_minute
end_total = end_hour * 60 + end_minute

travel_time = end_total - start_total

print("Travel time =", travel_time, "minutes")
