Power BI
========
Power BI is a business analytics and data visualization tool from Microsoft that helps you turn data into interactive reports and dashboards.

What Power BI is used for
📊 Creating charts, graphs, and dashboards
📈 Analyzing business performance and trends
🔄 Connecting to data sources such as Excel, SQL databases, cloud services, and APIs
🤝 Sharing reports across teams and organizations
🧠 Performing data modeling and advanced analytics


Main Components
--------------
> Power BI Desktop
Free Windows application for creating reports and data models.
> Power BI Service
Cloud-based platform for publishing, sharing, and viewing reports online.
> Power BI Mobile
Mobile apps for accessing dashboards on smartphones and tablets.

Download and Install 
==================
Download power BI Desktop from the Microsoft store as windows will automatically update your Power BI Desktop with the latest version.

(No need to download the entire application  for each update)

ex:
	Goto Microsoft store --> Search Power BI Desktop --> click to get button -->

Owner of E-commerce project wants us to help them to create a dashboard to track and analyze their online sales accross india.

step1:
-----
	Create two CSV files.
	ex:
		Details.csv or Details.xlsx
		Orders.csv or Orders.xlsx

step2:
-----
	Launch Power BI Dashboard.

	
step3:
-----
	Go to Power BI interface.
	ex:

step4:
-----
	Upload the data to Power BI.
	
	ex:
		click to Get Data (option) --> Text/CSV file --> select xlsx/csv file
		--> In Navigator select the sheet --> Load.
		
step5:
------	
	Check right side of POWER BI dashboard . It shows "Data" option.
