Iteration statement 
===================

In Python, an iteration statement is used to execute a block of code repeatedly. 

Python mainly provides two iteration statements:

1) for loop
2) while loop

1. for Loop (Definite Iteration)
-------------------------------
A for loop is used when you know how many times you want to repeat a block.
syntax:
------	
	for variable in sequence:
    		# code block

> Print numbers 1 to 5.
ex:

	for i in range(1, 6):
    		print(i)

> Iterating over a list
	
ex:
	fruits = ["apple", "banana", "cherry"]
	for fruit in fruits:
    		print(fruit)


> Sum of numbers

ex:
	total = 0

	for i in range(1, 6):
    	total += i

	print("Sum:", total)


> list using for loop
-------------------
A list is a collection of items that is:

Ordered
Mutable (can be changed)

ex:
numbers = [10, 20, 30, 40]

for num in numbers:
    print(num)

ex: Example with index
---
numbers = [10, 20, 30]

for i in range(len(numbers)):
    print("Index:", i, "Value:", numbers[i])


Tuple using for loop
-----------------------
A tuple is similar to a list but:

Ordered
Immutable (cannot be changed)

ex:

colors = ("red", "green", "blue")

for color in colors:
    print(color)

ex: Example with index:
ex:

colors = ("red", "green", "blue")

for i in range(len(colors)):
    print(i, colors[i])


> Dictionary using for loop

A dictionary stores data in key–value pairs.

ex:

#Iterate over keys
student = {"name": "John", "age": 20, "grade": "A"}

for key in student:
    print(key)

ex:

#Access values
for key in student:
    print(key, ":", student[key])


> Using .items() (best method)
ex:

for key, value in student.items():
    print(key, "=", value)


> Combined Example
data = {
    "numbers": [1, 2, 3],
    "colors": ("red", "blue")
}

for key, value in data.items():
    print("Key:", key)
    
    for item in value:
        print(item)



2. while Loop (Indefinite Iteration)
---------------------------
A while loop is used when the number of iterations is not known beforehand.

Syntax:
------
	while condition:
   	 	# code block

ex:

while True:
    print("Hello")

Use carefully; stop using Ctrl + C.


> Print numbers from 1 to 5

ex:
	i = 1
	while i <= 5:
   	 print(i)
    	i += 1
