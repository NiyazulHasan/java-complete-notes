Recursion 
=========
It is a programming technique which divides complex problem into smaller or sub-problems.

A function which call to itself for many number of times to solve a problem is called recursion.

Recursion is similar to loops. Whenever we use recursion we should not use loops.


Advantages of Recursion
-----------------------
Makes code shorter and cleaner for some problems.
Useful for:
Tree traversal
Graph algorithms
Divide-and-conquer algorithms
Dynamic programming problems

def print_numbers(n):
    if n > 10:
        return
    print(n)
    print_numbers(n + 1)

Q) Write a python program to find out factorial of a given number using recursion?

def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

num = int(input("Enter a number: "))

print("Factorial =", factorial(num))


Q) Write a python program to find out N-th element of fibonacci series?

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

n = int(input("Enter position: "))

print("Nth Fibonacci number =", fibonacci(n))


Q) Write a python program to find out fibonnaci series of a given number using recursion?

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

terms = int(input("Enter number of terms: "))

print("Fibonacci Series:")

for i in range(terms):
    print(fibonacci(i), end=" ")


Q) Write a python program to find out reverse of a given number using recursion?
def reverse_number(n, rev=0):
    if n == 0:
        return rev

    rev = rev * 10 + n % 10

    return reverse_number(n // 10, rev)

num = int(input("Enter a number: "))

print("Reverse =", reverse_number(num))
