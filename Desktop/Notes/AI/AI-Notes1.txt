AI - Artificial Intelligence
============================
It is a field of computer science which focuses on creating systems or machines that can perform multiple tasks typically required human intelligence.
ex:
	surveillance, Cars, Video games and etc.

It is a capability of a computer or a machine to mimic or simulate intelligence human behaviours such as thinking ,reasoning, perception, language understanding and etc.

It is a scientific discpline which embracing several data science fields ranging from Narrow AI to Strong AI which includes Machine learning, Deep learning, Generative AI , Data mining and etc.

Q) What is the difference between AI, ML, DL and Generative AI?

AI 
===
AI stands for Artificial Intelligence.
AI is designed to think and adopt the solutions.
ex:
	Google Maps, Face Recognition and etc.

ML
====
ML stands for Machine Learning.
ML is a subset of Aritifical Intelligence.
It is designed to learn from data instead of explicitly programmed.
ex:
	Spam calls, spam email deductions and etc.

DL 
====
DL stands for Deep Learning.
DL is a subset of Machine Learning.
It uses multi-layered neural networks to learn from large amount of data.
Here neural network means , it is one type of machine learning algorithm which is inspired by human brain.
ex:
	ChatGPT, Finger Prints and etc.

Generative AI 
=============
It is a subset of Deep Learning.
It is used to generate the data.
ex:
	Images, Videos, Audio, Text and etc.

AIController.java
-----------------
package com.ihub.www.controller;

import java.util.Map;

import com.ihub.www.ChatbotBackendApplication;
import org.springframework.ai.chat.client.ChatClient;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.CrossOrigin;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@CrossOrigin(origins = "http://localhost:5173")
@RestController
@RequestMapping("/api/ai")
public class AIController 
{

    private final ChatbotBackendApplication chatbotBackendApplication;
	private final ChatClient chatClient;
	
	AIController(ChatClient.Builder builder, ChatbotBackendApplication chatbotBackendApplication)
	{
		this.chatClient = builder.build();
		this.chatbotBackendApplication = chatbotBackendApplication;
	}
	
	@PostMapping("/ask")
	public ResponseEntity<String> ask(@RequestBody Map<String,String> payload)
	{
			String question = payload.get("question");
			String response = chatClient.prompt().user(question).call().content();
			return ResponseEntity.ok(response);
	}
}

App.jsx

import axios from "axios";
import { useState } from "react";
function App()
{
   const [question,setQuestion] = useState("");
   const [answer , setAnswer] = useState("");

   const handleClick= async ()=>{

    const response = await axios.post("http://localhost:9191/api/ai/ask",{question});
    setAnswer(response.data);
   }

    return (
      <div className="container mt-5">
      <h4 className="text-center"> Chat GPT </h4>
      <textarea name="address" cols={10} rows={8} className="form-control"
      placeholder="Ask something Here...."
      value={question}
      onChange={(e)=> setQuestion(e.target.value)}>
      </textarea>
      <button className="btn btn-primary mt-3 d-block mx-auto" onClick={handleClick}> Search </button>


      <div className="container mt-4">
        <p>
          {
              answer
          }
        </p>
      </div>  

      </div>
    )
}
export default App;
