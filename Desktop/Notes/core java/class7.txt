Identifiers 
===========
A name given to a program elements is called identifier.

Program elements are classes, interfaces, packages, variables, methods, constants and etc.

ex:
	class Test
	{
		public static void main(String[] args)
		{
			int x = 10;
			System.out.println(x); 
		}
	} 

Here Test, main, String, args, x, System, out and println are identifiers.

Rules to declare an identifiers
-------------------------------
Rule1:
------
	Identifier will accept following characters.
	ex:
		A-Z
		a-z
		0-9
		_
		$
Rule2:
-----
	If we take other characters then we will get compile time error.
	ex:
		int emp_id;
		int emp$alary;
		int dept#no;  // invalid 

Rule3:
------
	Every identifier is a case sensitive.
	ex:
		int number;
		int NUMBER;
		int NumBer;
Rule4:
------
	We can't take reserved words as an identifier.
	ex:
		int if;
		int else;
		int public;
		int class;

Rule5:
------
	Identifier must and should starts with alphabet, underscore or dollar symbol 
	but not with digit.
	ex:
		int  _abcd; 
		int  $alary; 
		int  a1234; 
		int  1abcd; // invalid 

Rule6:
------
	There is no length limit for an identifier but it is not recommanded 
	to take more then 15 characters.
