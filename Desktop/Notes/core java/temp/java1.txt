Q) Can we execute if and else simultenously?

Yes, it is possible to execute if and else simultenously using goto statement.

ex:
	void main()
	{	
		clrscr();
		if(0)
		{
			label1: printf("If Stmt");
			goto label2;
		}
		else
		{
			goto label1;
			label2: printf("Else Stmt");
		}
		getch();
	}
