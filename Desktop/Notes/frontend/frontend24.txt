JavaScript Spread Operator 
==========================
JavaScript spread operator (...) allows us to spread the array, to copy an array/object, modify an array/object, to merge an array/object.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
       <script>
          var arr = [10,20,30];

          function f1(arg1,arg2,arg3)
          {
               document.writeln(arg1+" "+arg2+" "+arg3);
          }

          f1(...arr);
       </script>
</body>
</html>

<!DOCTYPE html>

          var newArr = [...arr];

          for(var i in newArr)

Q) Explain Synchronous and Asynchronous in JavaScript?


Synchronous
------------
Synchronous JavaScript is a programming approach that enables the blocking execution of tasks, allowing sequence operations, degrading responsiveness, and non-efficient handling of time-consuming operations in web applications.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
       <script>
          document.writeln("stmt1");
          document.writeln("<br>");
          document.writeln("stmt2");
          document.writeln("<br>");
          document.writeln("stmt3");
       </script>

</body>
</html>

Asynchronous 
------------
Asynchronous JavaScript is a programming approach that enables the non-blocking execution of tasks, allowing concurrent operations, improved responsiveness, and efficient handling of no time-consuming operations in web applications.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
       <script>
          document.writeln("stmt1");
          document.writeln("<br>");
          
          setTimeout(function(){
            document.writeln("stmt2");
          },3000);

          document.writeln("<br>");
          document.writeln("stmt3");
       </script>

</body>
</html>


JavaScript hoisting
====================
Hoisting is a default behavior of JavaScript where all the variable and function declarations are moved on top. This means that irrespective of where the variables and functions are declared, they are moved on top of the scope. The scope can be both local and global.

JavaScript Set 
===============
A JavaScript Set is a collection of unique values. 
Each value can only occur once in a Set. 
The values can be of any type, primitive values or objects.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
       <script>
         
         var set = new Set();
         set.add(10);
         set.add(20);
         set.add(30);

         set.forEach(ele => document.writeln(ele));
          
       </script>

</body>
</html>

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
       <script>
         
         var set = new Set([10,20,30,40,50]);

         set.forEach(ele => document.writeln(ele));
          
       </script>

</body>
</html>

Q) Write a javascript program to display distinct elements?

input:
	1 2 2 3 3 3 4 4 4 4
output:
	1 2 3 4 

<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
       <script>
         var set = new Set();
         var arr = [1,2,2,3,3,3,4,4,4,4];
         for(var i in arr)
         {
            set.add(arr[i]);
         }
         set.forEach(ele => document.writeln(ele));
       </script>

</body>
</html>

===============

</html>
