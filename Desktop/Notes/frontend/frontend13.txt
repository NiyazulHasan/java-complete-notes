CSS animation property 
======================
CSS Animation property is used to create animation on the webpage.

CSS animation property is used to change the bhaviour of elements.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<style>
		div
		{
			width:200px;
			height:200px;
			background-color: #D980FA;
			margin:100px auto;
			border-radius: 50%;
			animation-name: circle;
			animation-duration: 5s;
			animation-iteration-count: infinite;
		}
		@keyframes circle
		{
			35%
			{
				background-color: #FFC312;
				border-radius: 0%;
			}
			75%
			{
				background-color: #ED4C67;
			}
		}
	</style>
</head>
<body>
		<div></div>
</body>
</html>

CSS Grid layout
===============
The CSS grid layout module offers a grid-based layout system with rows and columns.

Grid layout makes easier to design web pages without having a use of floats and positioning tag.

A grid layout consists of a parent element , with one or more child elements. 

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<style>
		.container
		{
			width:100%;
			height:450px;
			border:2px solid black;
		}
		.box1{background-color:#ff9ff3;}
		.box2{background-color:#feca57;}
		.box3{background-color:#ff6b6b;}
		.box4{background-color:#48dbfb;}
		.box5{background-color:#1dd1a1;}
		.box6{background-color:#5f27cd;}
	</style>
</head>
<body>
		<div class="container">
			<div class="box1">Box1</div>
			<div class="box2">Box2</div>
			<div class="box3">Box3</div>
			<div class="box4">Box4</div>
			<div class="box5">Box5</div>
			<div class="box6">Box6</div>
		</div>		
</body>
</html>
