Types of list in HTML
=====================
We have three types of list in HTML.

1) Unordered list 

2) Ordered list 

3) Description list 

1) Unordered list
-----------------
Unordered list is for bullets.
A <ul> tag is used to represent unordered list.
Unordered list contains list of items.
Each list of item we need to represent by using <li> tag.
ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		Courses:
		<ul>
			<li>MongoDB</li>
			<li>ExpressJS</li>
			<li>ReactJS</li>
			<li>NodeJS</li>
		</ul>	
	</body>
</html>

3) Description list 
--------------------
A description list contains description term and description definition.

A <dl> tag is used to represent description list.

A <dt> tag is used to represent description term.

A <dd> tag is used to represent description definition.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<dl>
			<dt>HTML5</dt>
			<dd>
				It is widely used language on web 
				to develop web pages and web 
				applications.
			</dd>
		</dl>
	</body>
</html>

HTML colors 
============
HTML supports only two color values.

1) Valid color name 
2) Hexa value 

1) Valid color name 
--------------------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body bgcolor="tomato">
		<font color="white">
			<h1> Heading Tag </h1>
		</font>		
	</body>
</html>

2) Hexa value
--------------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body bgcolor="#FFFF00">
		<font color="FF0000">
			<h1> Heading Tag </h1>
		</font>		
	</body>
</html>

HTML table 
===========
A table is used to represent the data.

A table is a collection of rows and columns.

A <table> tag is used to represent table in a web page.

A <tr> tag is used to represent table row.

A <th> tag is used to represent table heading.

A <td> tag is used to represent table data.

A table heading is used to display the text in bold and centered.

A table data is used to display the text in normal and left align.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>

HTML form
=========
HTML form is used to accept the data from the enduser.

It will forward the data to database or server for processing.

To declare a form we need to use <form> tag.

A HTML form contains following form components.
ex:
	Label 
	TextField
	PasswordField
	TextArea 
	radio button 
	checkbox 
	button 
	and etc.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<form>
			<label>UserName:</label>
			<input type="text" name="t1"/>  <br>
			
			<label>Password:</label>
			<input type="password" name="t2"/>  <br>
			
			<label>Age:</label>
			<input type="number" name="t3"/> 
		</form>
	</body>
</html>

Note:
-----
A "action" attribute describes where submitted data as to be transfer.

A "target" attribute describes submitted data must be open in a same window or new window.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<form action=""  target="_self">
			<label>UserName: </label>
			<input type="text" name="t1"/> <br>
			
			<label>Password:</label>
			<input type="password" name="t2"/> <br>
			
			<input type="submit" value="submit"/>
		</form>
	</body>
</html>
