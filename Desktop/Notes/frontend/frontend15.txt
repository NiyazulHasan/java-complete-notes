JavaScript History 
=================
Original name of JavaScript is LiveScript.

LiveScript developed by Netscape Corporation in 1991.

In 1995, Branden Eich a popular scientist of Netscape Corporation renamed LiveScript to JavaScript.

Official name of JavaScript is ECMA Script.

ECMA stands for European Company Manufacturer Association.

Output statements in JavaScript 
===============================
It is used to display data, variables, valid expressions and user defined statements.

We have two output statements in JavaScript.

1) document.write()

2) console.log() 

1) document.write()
--------------------
It is used to display the output on browser window.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		document.write("JavaScript Tutorial");
	</script>
</body>
</html>

2) console.log() 
--------------
It is used to display the output on browser console.
To see the browser console we need to use F12 function key.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		console.log("JavaScript Tutorial");
	</script>
</body>
</html>

Comments in JavaScript

JavaScript engine does not display the comments in output.

---------------------

	ex:

2) Multiple Line Comment 
-------------------------

	multiple lines.

		*/

Types of JavaScript 
===================
We have two types of JavaScript.

1) Internal JavaScript / Embedded JavaScript 

2) External JavaScript / Seperate JavaScript

1) Internal JavaScript
----------------------
In internal JavaScript we will declare HTML code and JavaScript code in .html file.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		document.write("Internal JavaScript");
	</script>
</body>
</html>

2) External JavaScript
----------------------
In external JavaScript we will declare HTML code in .html file and JavaScript code in .js file. We need to attach javascript file with HTML document by using <script> tag.

index.html 
----------
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<!-- add external JavaScript -->
	<script lang="text/javascript" src="myscript.js"></script>
</head>
<body>
	
		<button onclick="f1()"> ClickMe </button>

</body>
</html>

myscript.js
-----------
function f1()
{
	document.write("External JavaScript");
}


JavaScript variables 
=====================
A name given to a memory location is called variable.

Purpose of variable is used to store the data.

In JavaScript , variables are also know as identifiers.

We have same rules for variables as we have for identifiers.

Rules to declare variables
--------------------------
Rule1:
------
	A variable name starts with alphabet, underscore or dollar symbol.
	ex:		
		var a1234;
		var _abcd;
		var $alary; 

Rule2:
-----
	A variable name can't start with digit.
	ex:
		var 1abcd; //invalid 

Rule3:
------
	Every variable is a case sensitive.
	ex:
		var a;
		var A;

We have two types of variables in JavaScript.

1) Local variable

2) Global variable 

1) Local variable
------------------
If we declare any variable inside a function is called local variable.

ex:
--
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		function f1()
		{
			//local variable 
			var i = 100;
			document.write(i);
		}

		//calling 
		f1();
	</script>
</body>
</html>

Local variable we can access within function only.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		function f1()
		{
			//local variable 
			var i = 100;
			document.write(i);
		}
		function f2()
		{
			document.write(i);
		}

		//calling 
		f1();
		f2(); // Error i is not defined
	</script>
</body>
</html>

2) Global variable 
-------------------
If we declare any variable outside the function is called global variable.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		// global variable 
		var i = 200;

		function f1()
		{
			document.writeln(i);
		}

		//calling 
		f1();
	</script>
</body>
</html>

Global variable we can access within the function and outside the function.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		// global variable 
		var i = 200;

		function f1()
		{
			document.write(i)
		}
		function f2()
		{
			document.write(i)
		}

		//calling 
		f1();
		f2();
	</script>
</body>
</html>
