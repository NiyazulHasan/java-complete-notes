3) External CSS
================
In external CSS, we will declare HTML code in .html file and CSS code in .css file.

We can't execute CSS file directly to the browser window.

To execute CSS file we need to link that file with HTML document.

index.html
----------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<!-- add external css -->
		<link rel="stylesheet" href="css/mystyles.css">
	
	</head>
	<body>
			<h1>Welcome to CSS class</h1>
			<p> This is External CSS </p>
	</body>
</html>

mystyles.css
------------
h1
{
	color:#FFFFFF;
	background-color:#079992;
}
p
{
	font-size:40px;
	text-align:center;
}

CSS margin property 
===================
CSS margin property is used to specify space around the elements(outside defined border).

We have following list of CSS margin properties.

ex:
	margin-top 
	margin-right 
	margin-bottom 
	margin-left 
	margin shorthand 

We have following list of values for margin property.
ex:
	value			Description 
	-------			-----------
	auto 			Browser calculates the margin 
	length 			It specifies the margin in px,pt,cm,em and etc.
	percentage 		It specifies the margin in percentage(%)
	inherit 		It specified the margin from parent element.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border:2px solid black;
				margin-top:100px;
				margin-right:100px;
				margin-bottom:100px;
				margin-left:100px;
			}
		</style>
	</head>
	<body>
			<h1> Heading Tag 1</h1>

	</body>
</html>

CSS Height and Width property 
=============================
CSS height property is used to set the height of HTML element.
CSS width property is used to set the width of HTML element.
CSS height and width property does not include border, margin and padding.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			div
			{
				border:2px solid red;
				height:200px;
				width:200px;
			}
		</style>
	</head>
	<body>
		<div>
			Division Tag
		</div>
	</body>
</html>
