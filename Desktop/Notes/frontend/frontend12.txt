CSS z-index property 
====================
The z-index property specifies the stack order of an element.

An element with greater stack order is always in front of an element with a lower stack order.

Note: z-index only works on positioned elements.
	

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<style>
		img
		{
			width:200px;
			height:200px;
			position: absolute;
			left:0;
			top:0;
			z-index: 1;
		}
	</style>
</head>

CSS Google Fonts
=================
If we do not want to use any of the standard fonts in HTML, you can use Google Fonts.

Google Fonts are free to use, and have more than 1000 fonts to choose.


To use any google fonts we need to use below url.
ex:
	https://fonts.google.com/
ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<link href="https://fonts.googleapis.com/css2?family=Jacquard+24&display=swap" rel="stylesheet">
	<style>
		h1
		{
			text-align: center;
			font-family: "Jacquard 24", system-ui;
		}
	</style>
</head>
<body>
	<h1> Welcome to 2026 to Build Bright Future </h1>
</body>
</html> 

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>

	<style>
	@import url('https://fonts.googleapis.com/css2?family=Jacquard+24&display=swap');
		h1
		{
			text-align: center;
			font-family: "Jacquard 24", system-ui;
		}
	</style>
</head>
<body>
	<h1> Welcome to 2026 to Build Bright Future </h1>
</body>

CSS Fontawesome Icons 
=====================

CSS Gradients
==============
CSS gradients let you display smooth transitions between two or more specified colors.

CSS defines three types of gradients:

1)Linear Gradients (goes down/up/left/right/diagonally)

2)Conic Gradients (rotated around a center point)

3)Radial Gradients (defined by their center)
		 

1) Linear Gradient
------------------
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<style>
		body
		{
			height: 100vh;
			background : linear-gradient(yellow,red);
		}
	</style>
</head>
<body>
		
</body>
</html>

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>

		body
		{
			height: 100vh;
			background : linear-gradient(to top,yellow,red);
		}
	</style>
</head>
<body>
		
</body>
</html>

CSS flexbox
=============
Flexbox is a one-dimensional layout methods for laying out items in rows and columns.

CSS flexbox is a better way to align items into a container.

Flexbox= flexible + box.

To create a flexbox model, we need to define a "flex-container".
ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<style>
		.container 
		{
			width:100%;
			height:450px;
			border:2px solid black;
			box-sizing: border-box;
			display: flex;
			flex-direction: row;
		}
		.container .item
		{
			width:150px;
			height:150px;
			border:2px solid red;
			box-sizing: border-box;
		}
	</style>
</head>
<body>
		<div class="container">
			<div class="item">Item1</div>
			<div class="item">Item2</div>
			<div class="item">Item3</div>
			<div class="item">Item4</div>
			<div class="item">Item5</div>
			<div class="item">Item6</div>
		</div>
</body>
</html>

CSS cursor property 
===================
The cursor property specifies the mouse cursor to be displayed when pointing over an element.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!!</title>
	<style>
		img
		{
			width:300px;
			height:300px;
			display: block;
			margin: 100px auto;
			border:2px solid black;
			cursor : pointer;
		}
	</style>
</head>
<body>
		<img src="images/loki.png"/>
</body>
</html>
