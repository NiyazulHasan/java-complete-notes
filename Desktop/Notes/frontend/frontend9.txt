CSS overflow property
========================
The overflow property specifies what should happen if content overflow.

This property spcifies weither to clip content or to add scrollbars  when an element content is too big to fit in a specified area.

Note:
---------
The overflow property works for block elements with  a specified height.

value		Description
---------	-----------------
visible		The overflow is not clipped.It rendered outside the element 's box and 
		it is default value.

hidden		the overflow is clipped  and rest of the content will be invisible.

scroll		The overflow is clipped,but a scroll-bar is added to see the rest of the 
		content .

auto		The overflow is clipped,a scroll-bar should be added to the rest of the 
		content.


ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			div
			{
					width:200px;
					height:200px;
					border:2px solid black;
					overflow:visible;			
			}

		</style>
	</head>
	<body>

				from browsing to online shopping and banking.enabling 
				communication and content delivery over the internet for activities
				from browsing to online shopping and banking.

CSS image shadow property
===========================
A drop shadow is effectively a blurred, offset version of the input image's alpha mask, drawn in a specific color and composited below the image.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			img
			{
				width:300px;
				height:300px;
				margin: 150px 500px;
				filter:drop-shadow(18px 12px #C3C3C3);
			}

		</style>
	</head>
	<body>
		<img src="images/micky.png"/>
	</body>
</html>

Types of selectors in CSS
=========================
We have five selectors in CSS.

1) Element selector 
2) Group selector 
3) Id selector 
4) class selector 
5) Universal selector 

1) Element selector 
-------------------
The element selector selects HTML elements based on element name.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				color:#6F1E51;
				text-align:center;
			}

		</style>
	</head>
	<body>
		<h1> Heading Tag </h1>
		<h1> Heading Tag </h1>
		<h1> Heading Tag </h1>
	</body>
</html>


2) Group Selector 
-----------------
The grouping selector selects all the HTML elements with the same style definitions.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1,p,div
			{
				color:#6F1E51;
				text-align:center;
				background-color:#D980FA;
			}

		</style>
	</head>
	<body>
		<h1> Heading Tag </h1>
		<p> Paragraph Tag </p>
		<div> Division Tag </div>
	</body>
</html>


3) Id Selector 
---------------
The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element is unique within a page, so the id selector is used to select one unique element.

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			#myId1 
			{
				color : #B53471;
			}
			#myId2
			{
				background-color : #0652DD;
			}
		</style>
	</head>
	<body>
		<h1 id="myId1"> Heading Tag1 </h1>
		<h1 id="myId2"> Heading Tag2 </h1>
		
	</body>
</html>

One element can use only one ID at a time.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			#myId1 
			{
				color : #B53471;
			}
			#myId2
			{
				background-color : #0652DD;
			}
		</style>
	</head>
	<body>
		<h1 id="myId1"> Heading Tag1 </h1>
		<h1 id="myId2"> Heading Tag2 </h1>
		<h1 id="myId1 myId2"> Heading Tag3 </h1>
	</body>
</html>


4) Class Selector
------------------
The class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period (.) character,
followed by the class name.

ex:
---

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			.myClass1
			{
				color : #B53471;
			}
			.myClass2
			{
				background-color : #0652DD;
			}
		</style>
	</head>
	<body>
		<h1 class="myClass1"> Heading Tag1 </h1>
		<h1 class="myClass2"> Heading Tag2 </h1>
		
	</body>
</html>

One element can use multiple classes.

ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			.myClass1
			{
				color : #B53471;
			}
			.myClass2
			{
				background-color : #0652DD;
			}
		</style>
	</head>
	<body>
		<h1 class="myClass1"> Heading Tag1 </h1>
		<h1 class="myClass2"> Heading Tag2 </h1>
		<h1 class="myClass1 myClass2"> Heading Tag3 </h1>
	</body>
</html>



5) Universal Selector 
----------------------
The universal selector (*) selects all HTML elements on the page.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			*
			{
				color : #FF0000;
				text-align:center;
			}
		</style>
	</head>
	<body>
		<h1> Heading Tag </h1>
		<p> Paragraph Tag </p>
		<div> Division Tag </div>
	</body>
</html>
