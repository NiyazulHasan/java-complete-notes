HTML Basic Tags 
===============

Heading Tags 
------------
It is used to display headings in our web page.

We have six heading tags i.e h1 to h6.

It is used to display the text in bold and size of the text is depend upon number of heading tags.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<h1> Heading Tag 1 </h1>
		<h2> Heading Tag 2 </h2>
		<h3> Heading Tag 3 </h3>
		<h4> Heading Tag 4 </h4>
		<h5> Heading Tag 5 </h5>
		<h6> Heading Tag 6 </h6>
		
	</body>
</html>

Paragarph tag 
-------------
A <p> tag is used to display the paragraph in a web page.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<p>
			HTML is widely used language on web to develop web pages
			and web applications. It is used to develop static web 
			pages only and all HTML documents executed in a browser.
		</p>		
	</body>
</html>

Q) What is the difference between <b> and <strong> tag ?

<b>
----
A bold tag is used to display the text in bold without importance.
ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<b>It is Bold Tag</b>	
	</body>
</html>


<strong>
--------
A strong tag is used to display the text in bold with importance.
ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<strong>It is Strong Tag</strong>	
	</body>
</html>

Nested Tags in HTML
===================
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<center>
			<h1>
				<font color="blue" size="18" face="serif">
					Welcome to HTML5
				</font>
			</h1>
		</center>
	</body>
</html>


Phrase Tags 
===========
Phrase tags are special purpose tags because they define special meaning to text or block of code.

We have following list of phrase tags in HTML5.

1) Abbreviation - <abbr> 
2) blockquote   - <blockquote> 
3) address 	- <address>
4) code		- <code> 
5) preformatted tag - <pre> 
6) variable 	- <var>
7) sample	- <samp> 
8) short quote	- <q>
9) strike	- <s> 
10) cite	- <cite> 
11) mark	- <mark>
12) subscript	- <sub>
13) superscript	- <sup> 
14) keyboard	- <kbd> 
15) bi directional order - <bdi> 
and etc.
