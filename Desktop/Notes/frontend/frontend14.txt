CSS Pseudo Elements ::before ::after 
================================
A CSS pseudo-element is used to style specific parts of an element.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		h1
		{
			text-align: center;
		}
		h1::before
		{
			content:"Ihub";
			color:#FF0000;
		}
		h1::after
		{
			content:"Institute";
			color:#FF0000;
		}
	</style>
</head>
<body>
	<h1>Talent</h1>
</body>
</html>

ex:

Global variable in CSS
======================
In CSS, global variables (officially called custom properties) are declared within the :root pseudo-class selector. This selector targets the highest-level element in the document (the <html> tag), ensuring the variables are available and can be accessed throughout the entire document via the var() function. 

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	<style>
		:root
		{
			--purple-color:#D980FA;
			--mahroon-color:#B53471;
			--blue-color:#1B1464;
			--text-center:center;
		}
		body
		{
			background-color: var(--purple-color);
		}	
		h1
		{
			color : var(--mahroon-color);
			text-align: var(--text-center);
		}
		p
		{
			color : var(--blue-color);
		}
	</style>
</head>
<body>
		<h1> Heading Tag </h1>
		<p> Paragraph Tag</p>
</body>
</html>

=========
Forms are used to accept the inputs from enduser.

Form forwards the data to database or server for processing.
