JavaScript Date 
===============
JavaScript Date object is used to display date and time.

Using JavaScript Date object we can display timer on the web page.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
     <script>
        var d = new Date();
        document.writeln(d);
     </script>
</body>
</html>

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
     <script>
        var date = new Date();
        var d = date.getDate();
        var m = date.getMonth()+1;
        var y = date.getFullYear();
        document.writeln(d+"/"+m+"/"+y);
     </script>
</body>
</html>

Q) What is difference between innerText and innerHTML property?

innerText 
---------
It is used to display simple text in javascript dynamically.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
         <div id="result"></div>
         <script>
            
            document.getElementById('result').innerText ="<p> innerText Property </p>";

         </script>
</body>
</html>

innerHTML
---------
It is used to display simple text and HTML code in javascript dynamically.

ex:
--
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
         <div id="result"></div>
         <script>
            
            document.getElementById('result').innerHTML ="<p style='color:red'> innerText Property </p>";

         </script>
</body>
</html>

Higher Order Function 
======================
Function which operates on another function by taking as an argument or by returning a function is called higher order function.

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
        <script>
           function f1(f2)
           {
               f2();
           }
           f1(function(){document.writeln("Higher Order Function 1")});
        </script>
</body>
</html>

ex:
---
<!DOCTYPE html>
<html>
<head>
   <title>MyPage!</title>
</head>
<body>
        <script>
           function f1()
           {
            return function(){document.writeln("Higher Order Function 2");}
           }
           var f2 = f1();
           f2();
        </script>
</body>
</html>
