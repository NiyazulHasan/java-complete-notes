Bootstrap tables
================
A table is a collection of rows and columns.
A table is used to represent the data in the form of rows and columns.
To create a bootstrap table we need to use .table class.
To create a border in a table we need to use .table-bordered class.
To remove the border from a table we need to use .table-borderless class.
To create a striped table we need to use .table-striped class.

ex:
---
<!DOCTYPE html>
<html>
   <head>
      <title>MyPage!</title>
     
      <!-- Bootstrap plugins -->
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
     
      <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">

      <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>

   </head>
   <body>
            <div class="container">
                 <table class="table">
                    <tr>
                       <th>No</th>
                       <th>Name</th>
                       <th>Address</th>
                    </tr>
                    <tr>
                       <td>101</td>
                       <td>Alan</td>
                       <td>Texas</td>
                    </tr>
                    <tr>
                       <td>102</td>
                       <td>Jose</td>
                       <td>Florida</td>
                    </tr>
                    <tr>
                       <td>103</td>
                       <td>Nelson</td>
                       <td>Chicago</td>
                    </tr>
                    <tr>
                       <td>104</td>
                       <td>Kelvin</td>
                       <td>Vegas</td>
                    </tr>
                 </table>
            </div>
   </body>
</html>

Bootstrap Alerts 
================
To create a bootstrap alert we need to use .alert class along with contextual classes.

We have following lists of contextual classes.

ex:
	alert-danger 
	alert-success
	alert-info 
	alert-warning 
	and etc

ex:
---
<!DOCTYPE html>
<html>
   <head>
      <title>MyPage!</title>
     
      <!-- Bootstrap plugins -->
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
     
      <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">

      <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>

   </head>
   <body>
            <div class="container mt-5">
                 <div class="alert alert-warning">
                     <strong>Warning!!</strong> Please attend all the classes 
                 </div>

                 <div class="alert alert-info">
                     <strong>Info!!</strong>If u prepare well you will get job 
                 </div>

                 <div class="alert alert-success">
                     <strong>success!!</strong> I got the job 
                 </div>
                  <div class="alert alert-danger">
                     <strong>Danger!!</strong> No practice No Job
                 </div>
            </div>
   </body>
</html>

Bootstrap list 
==============
A .list-group class is used to declare a bootstrap list.

A .list-group-item class is used to declare a bootstrap list of item.

We have following classes to add the color to bootstrap list of item.
ex:
	.list-group-item-primary 
	.list-group-item-secondary
	.list-group-item-info
	.list-group-item-warning
	.list-group-item-danger
	.list-group-item-success 
	and etc.

ex:
------

                     <li class="list-group-item">JavaScript</li>
                     <li class="list-group-item">Bootstrap</li>
                 </ul>

            </div>
   </body>
</html>

Bootstrap Badge
=================
A .badge class is used to declare bootstrap badge.

It is used to highlights the notifications.

It is used to add additional information to any content.


ex:
---

                 <button class="btn btn-primary">
                    Email <span class="badge bg-light text-primary">5</span>
                 </button>

            </div>
   </body>
</html>

Bootstrap Buttons 
=================

We have following list of contextual classes.
ex:

	.btn-secondary 
	.btn-info 
	.btn-warning 
	.btn-dark
	.btn-light
	.btn-success
	.btn-danger 
	or 
	.btn-outline-primary

	.btn-outline-info 
	.btn-outline-warning 
	.btn-outline-dark

Bootstrap Progress Bar 
======================
A progress bar is a visual component that indicates the progress of a task or process.
ex:
	downloading a file 
	installing a software 
	transfering a data 
	and etc. 

We use the .progress as a wrapper to indicate the max value of the progress bar.

We use the inner .progress-bar to indicate the progress so far.

The .progress-bar requires an inline style, utility class, or custom CSS to set their width.


ex:
----
<!DOCTYPE html>
<html>
   <head>
      <title>MyPage!</title>
     
      <!-- Bootstrap plugins -->
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
     
      <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">

      <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" ></script>

   </head>
   <body>
            <div class="container mt-5">

               <div class="progress">
                  <div class="progress-bar" style="width:50%"> 50% </div>   
               </div>

            </div>
   </body>
</html>
