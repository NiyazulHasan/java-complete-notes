4) Logical Operators 
---------------------

ex:
	Operator	Description 
	--------	-----------
	&&		Logical AND 
	||		Logical OR

Logical AND (&&)
----------------
Logical ANd operator deals with boolean values either true or false.

Truth table 
-----------
T	T	= T
T	F	= F 
F	T	= F 
F	F	= F 

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
	document.writeln(((5>2) && (6<10))+"<br>");

        document.writeln(((5>20) && (6<10))+"<br>");

        document.writeln(((5>20) && (6<1))+"<br>");
	</script>
</body>
</html>

Logical AND (||)
----------------
Logical OR operator deals with boolean values either true or false.

Truth table 
-----------
T	T	= T
T	F	= T 
F	T	= T 
F	F	= F 

ex:

<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		document.writeln(((5>2) || (6<10))+"<br>"); //true 

        document.writeln(((5>20) || (6<10))+"<br>"); //true 

        document.writeln(((5>20) || (6<1))+"<br>"); // false 
	</script>
</body>
</html>

3) IF ELSE IF stmt 
------------------
It evaluates the code based on multiple conditions.

syntax:
-------
	if(cond1)
	{
		- //code to be evaluate if cond1 is true 
	}
	else if(cond2)
	{
		- //code to be evaluate if cond2 is true 
	}
	else if(cond3)
	{
		- //code to be evaluate if cond3 is true 
	}
	else
	{
		- //code to be evaluate if all conditions are false
	}

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		var n = parseInt(prompt("Enter the option :"));//
        if(n==100)
            document.writeln("It is a police number");
        else if(n==103)
            document.writeln("It is enquiry number");
        else if(n==108)
            document.writeln("It is emergency number");
        else
            document.writeln("Invalid option");
	</script>
</body>
</html>

JavaScript Switch 
==================
It evaluates the code based on multiple conditions.
It is similar to if else if stmt.
A switch statement is used to read options from the enduser.

syntax:
-------
	switch(condition/expression)
	{
		case  value1: //code to be evaluate 
			      break stmt;

		case  value2: //code to be evaluate 
			      break stmt;
		-
		-
		default : //code to be evaluate if all cases are false.

	}

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		var n = parseInt(prompt("Enter the option :"));//
        switch(n)
        {
            case 100 : document.writeln("It is police number");
                        break;

            case 103 : document.writeln("It is enquiry number");
                        break;

            case 108 : document.writeln("It is emergency number");
                        break;

            default : document.writeln("Invalid option");
        }
	</script>
</body>
</html>

Note:
-----
Declaration of break stmt is optional. Suppose if we won't declare break statement then from where our condition is satisfied from there all cases will be executed.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
</head>
<body>
	<script>
		var n = parseInt(prompt("Enter the option :"));//
        switch(n)
        {
            case 100 : document.writeln("It is police number");
                        //break;

            case 103 : document.writeln("It is enquiry number");
                        //break;

            case 108 : document.writeln("It is emergency number");
                        //break;

            default : document.writeln("Invalid option");
        }
	</script>
</body>
</html>
