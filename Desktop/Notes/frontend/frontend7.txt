CSS
====
CSS stands for Cascading Styles Sheet.

Like HTML, It is widely used language on web.

The latest version of CSS3 was released in the year of 2001.

The main objective of CSS are 

1) To set the position of an elements.

2) To apply styles on elements to describe who an elements are look like.

3) To perform some sort of animations.

Advantages
==========
1) It is easy to learn and easy to use.

2) Good Performance.

3) It is used to apply the styles on elements.

4) It supported by all major browsers.

5) It supports global change.

CSS example
===========
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<style type="text/css">
			-
			-  CSS properties 	
			-	
		</style>
	</head>
	<body>
		
	</body>
</html>

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style type="text/css">
			h1
			{
				color:blue;
				background-color:cyan;
				text-align:center;
			}			
		</style>
	</head>
	<body>
			<h1> Heading Tag </h1>
	</body>
</html>

Note:
-----
	Here styles cascade from head to body.



CSS syntax
==========
CSS rule set contains element selector and declaration block.
ex:
	selector 
	|
	h1{color:blue;background-color:cyan;text-align:center;}	
	  |---------------------------------------------------|
				  |
			declaration block 

Element selector describes to which element we need to apply the styles. 	
declaration block contains list of CSS properties.
Each CSS property contains property name and property value seperated with colon.
Each CSS property ends with semicolon.


CSS background property 
======================
CSS background property is used to set the background in a web page.

If we set the background to body then it applies to entire web page.

We can set background to a perticular element.
ex:
	<h1>, <div>, <p>, <b> and etc.

We have following list of CSS background properties.

1) background-color 

2) background-image 

3) background-repeat

4) background-size 

5) background-position 

6) background attachment 

7) background shorthand 

8) background blend mode 

and etc.

Types of CSS
=============
We have three types of CSS.

1) Inline CSS 

2) Internal CSS 

3) External CSS 


1) Inline CSS
--------------
It is used to apply unique style on single element.

Using "style" attribute we can achieve inline CSS.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<h1 style="color:#FDA7DF;background-color:#6F1E51;text-align:center;">
			Heading Tag 1
		</h1>
			
		<h1> Heading Tag 2 </h1>
	</body>
</html>

2) Internal CSS 
-----------------
It is used to apply unique styles on a single web page.

Using <style> tag we can achieve internal CSS.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				color : #EA2027;
				background-color:#D980FA;
				text-align:center;
			}
		</style>
	</head>
	<body>

			<h1> Heading Tag 1 </h1>
			<h1> Heading Tag 2 </h1>
	</body>
</html>
