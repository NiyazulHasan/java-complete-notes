PL/SQL 
=======
PL/SQL stands for Procedural Language Extensions to Structured Query Language.

Using PL/SQL we can achieve following features.

1) We can achieve programming features like control statements, loops and etc.

2) It reduces network traffic.

3) We can display custom messages by using concept called exception handling.

4) We can perform related operations by using triggers.

5) It stores the program permanently to database for repeated execution.

and etc.


PL/SQL BLOCK 
============
A PL/SQL block is also known as PL/SQL program.

ex:
	DECLARE
	-
	-  -- Declaration Section 
	-
	BEGIN
	-
	-  -- Executable Section 
	-	
	EXCEPTION
	-
	-  -- Exception Section 
	-	
	END;
	/


Declaration Section
--------------------
It is used to declare variables, exceptions, cursors and etc.
It is optional section.

Executable Section 
------------------
It contains lines of code which is used to complete a table.
It is a mandatory section.

Exception Section 
----------------
It contains lines of code which is executed when exception is raised.
It is a optional section.

To see the output in PL/SQL we need to use below command.
ex:
	set  serveroutput  on 



Q) Write a PL/SQL program to display Hello World?

	BEGIN 
	DBMS_OUTPUT.PUT_LINE('Hello World');
	END;
	/ 

	Here '/' is used to submit the program in database.


Q) Write a PL/SQL program to perform sum of two numbers?

	DECLARE 
	A number;
	B number;
	C number;
	BEGIN
	A:=10;
	B:=20;
	C:=A+B;
	DBMS_OUTPUT.PUT_LINE(C);
	END;
	/

Declaration and Initialization in a single line 
--------------------------------------------

	DECLARE 
	A number := 10;
	B number := 20;
	C number := A+B;
	BEGIN 
	DBMS_OUTPUT.PUT_LINE('Sum of two numbers is ='||C);
	END;
	/

	Here DBMS_OUTPUT is a package name. 
	Here PUT_LINE is a procedure name.



In PL/SQL DML operations are allowed.

Q) Write a PL/SQL program to insert a record into student table?

	DECLARE 
	L_sno  number(3);
	L_sname varchar2(10);
	L_sadd varchar2(12);
	BEGIN 
	L_sno := &sno;
	L_sname := '&sname';
	L_sadd := '&sadd';
	insert into student values(L_sno,L_sname,L_sadd);
	DBMS_OUTPUT.PUT_LINE('Record Inserted');
	END;
	/	

Q) Write a PL/SQL program to update student name based on student number?

	DECLARE 
	L_sno number(3);
	BEGIN 
	L_sno := &no;
	update student set sname='jessica' where sno=L_sno;
	DBMS_OUTPUT.PUT_LINE('Record Updated');
	END;
	/	

Q) Write a PL/SQL program to delete student record based on student number?

	DECLARE 
	L_sno number(3);
	BEGIN 
	L_sno := &sno;
	delete from student where sno=L_sno;
	DBMS_OUTPUT.PUT_LINE('Record Deleted');
	END;
	/

In PL/SQL DRL operations are allowed.

To execute select command we need to use "into" clause.


Q) Write a PL/SQL to select employee name whose employee id is 201?


	DECLARE 
	L_ename varchar2(10);
	BEGIN 
	select ename into L_ename from emp where eid=201;
	DBMS_OUTPUT.PUT_LINE(L_ename);
	END;
	/


Q) Write a PL/SQL program to display employee name, employee salary whose 
   employee id is 202?

	DECLARE

Percentage(%) TYPE Attribute
============================
It is used to declare a local variable which respect to column type.

We can declare percentage type attribute as follow.
ex:
	<variable_name>  <table_name>.<col_name>%TYPE;



Q) Write a PL/SQL program to display employee name, employee salary whose 
   employee id is 202?

	DECLARE 
	L_ename emp.ename%TYPE;
	L_esal  emp.esal%TYPE;
	BEGIN 
	select ename,esal into L_ename,L_esal from emp where eid=202;
	DBMS_OUTPUT.PUT_LINE(L_ename||' '||L_esal);
	END;
	/


Percentage(%) ROWTYPE attribute 
===============================
It is used to declare a local variable which holds complete row of a table.

ex:
	<variable_name>  <table_name>%ROWTYPE;

ROWTYPE variable we can't print directly.

To get the values from RWOTYPE variable we need to use 
ex:
	<variable_name>.<column_name> 


Q) Write a PL/SQL program to display employee information whose employee id is 204?

	DECLARE 
	A emp%ROWTYPE;
	BEGIN 
	select * into A from emp where eid=204;
	DBMS_OUTPUT.PUT_LINE(A.eid||' '||A.ename||' '||A.esal||' '||A.deptno||' '||A.job||' '||A.comm);
	END;
	/


Q) Write a PL/SQL program to display employee information based on employee id?

	DECLARE 
	L_eid emp.eid%TYPE;
	A emp%ROWTYPE;
	BEGIN 
	L_eid := &eid;
	select * into A from emp where eid = L_eid;
	DBMS_OUTPUT.PUT_LINE(A.eid||' '||A.ename||' '||A.esal||' '||A.deptno);
	END;
	/

Control statements
==================
We have three control statements in PL/SQL.

1) IF THEN 

2) IF THEN ELSE 

3) IF THEN ELSIF THEN ELSE 

1) IF THEN
-----------
It evalutes the code only if our condition is true.

ex:
	DECLARE 
	A number:=100;
	BEGIN 
	
	IF A>50 THEN 
	DBMS_OUTPUT.PUT_LINE('A is greatest');
	END IF;

	END;
	/

ex:
	DECLARE 
	A number:=100;
	BEGIN 
	
	IF A>500 THEN 
	DBMS_OUTPUT.PUT_LINE('A is greatest');
	END IF;

	END;
	/

2) IF THEN ELSE 
------------
It evalutes the code either our condition is true or false.

ex:
	DECLARE 
	A number:=100;
	BEGIN 
	
	IF A>50 THEN 
	DBMS_OUTPUT.PUT_LINE('A is greatest');
	ELSE
	DBMS_OUTPUT.PUT_LINE('A is smallest');
	END IF;
	
	END;
	/

ex:
	DECLARE 
	A number:=10;
	BEGIN 
	
	IF A>50 THEN 
	DBMS_OUTPUT.PUT_LINE('A is greatest');
	ELSE
	DBMS_OUTPUT.PUT_LINE('A is smallest');
	END IF;
	
	END;
	/

3) IF THEN ELSIF THEN ELSE 
--------------------------
It evaluates code based on multiple conditions.

ex:
	DECLARE 
	A number;
	BEGIN 
	A:=&opt;
	IF A=100 THEN
	DBMS_OUTPUT.PUT_LINE('It is a police number');
	ELSIF A=103 THEN 
	DBMS_OUTPUT.PUT_LINE('It is a enquiry number');
	ELSIF A=108 THEN 
	DBMS_OUTPUT.PUT_LINE('It is a emergency number');
	ELSE
	DBMS_OUTPUT.PUT_LINE('Invalid option');
	END IF;
	END;
	/



PL/SQL LOOPS 
=============
Loops are used to evalute the code repeatedly.

We have three types of loops in PL/SQL.

1) Simple Loop 

2) While Loop

3) For Loop

1) Simple Loop 
-------------
It evalutes the code how long our condition is true.

ex:
	DECLARE 
	A number:=1;
	BEGIN 
	DBMS_OUTPUT.PUT_LINE('Welcome');
	
	LOOP
	DBMS_OUTPUT.PUT_LINE('Hello');
	EXIT WHEN A=4;
	A:=A+1;
	END LOOP;

	DBMS_OUTPUT.PUT_LINE('Thank You');
	END;
	/

ex:
---
	DECLARE 
	A number:=1;
	BEGIN 
	
	LOOP 
	DBMS_OUTPUT.PUT_LINE(A);
	EXIT WHEN A=10;
	A:=A+1;
	END LOOP;
	
	END;
	/

2) While Loop
---------------
It evaluates the code how long our condition is true.

ex:
	DECLARE 
	A number:=1;
	BEGIN 
	DBMS_OUTPUT.PUT_LINE('Welcome');

	WHILE A<=4 LOOP 
	DBMS_OUTPUT.PUT_LINE('Hello');
	A:=A+1;
	END LOOP;

	DBMS_OUTPUT.PUT_LINE('Thank You');
	END;
	/

ex:
---
	DECLARE 
	N number;
	A number:=1;
	BEGIN 
	N:=&no;
	
	WHILE A<=10 LOOP
	DBMS_OUTPUT.PUT_LINE(N||' * '||A||' = '||N*A);
	A:=A+1;
	END LOOP;

	END;
	/

3) For Loop
-------------
It is used to evalute the code how long our condition is true.

ex:
	DECLARE 
	A number;	
	BEGIN 
	DBMS_OUTPUT.PUT_LINE('Welcome');
	
	FOR A IN 1 .. 4 LOOP
	DBMS_OUTPUT.PUT_LINE('Hello');
	END LOOP;

	DBMS_OUTPUT.PUT_LINE('Thank You');
	END;
	/



ex:
	DECLARE 
	A number;	
	BEGIN 
	
	FOR A IN 10 .. 1 LOOP
	DBMS_OUTPUT.PUT_LINE(A);
	END LOOP;

	END;
	/
