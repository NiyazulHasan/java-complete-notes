Spring Data JPA 
===============
Spring Data JPA is also known Data Access Framework.

It is a sub module of spring framework which interacts with relational databases in java application using JPA (Java Persistence API).

It is a part of a spring framework that helps to communicate with database using objects instead of writing complex queries.

ex:
	To fetch the data we need to write below query.
	ex:
		select * from student;

	In Spring Data JPA , we need to use below method.
	ex:
		findAll();

It removes boiler plate code by providing some common database operations like findById(), delete(), save() and etc.

It allows us to declare only methods , Spring Data JPA writes the query for us.

It works with JPA provider like Hibernate to handle all Object Relational Mapping.

It provides several repositories , which we can extends in our project to perform database operations , paging , sorting and etc.

We have following core repositories which are commonly used in spring data jpa.

1) Repository <I,ID> 
-------------------
> It is a basic repository for all repositories.
> It is not itended to use.


2) CrudRepository<I,ID>
----------------------
> It extends Repository.
> It is used to perform database operations.
  ex:
	findById(), findAll(), save(), delete() and etc.

3) PagingAndSortingRepository<I,ID> 
------------------------------------
> It extends CrudRepository.
> It is used to perform database operations and it add's paging and sorting features.
  ex:
	findAll(Pageable pageable)
	findAll(Sort sort)
	and etc.
		
4) JpaRepository<I,ID>
---------------------
> It is commonly used repository in spring data jpa.
> It extends CrudRepository and PagingAndSortingRepository.
> It contains JPA specific methods.
  ex:
	flush()
	saveAndFlush(S entity)
	deleteAllInBatch()
	and etc.

step3:
------
	Create a "Item.java" file inside "com.ihub.www.entity" package.

Item.java
---------
package com.ihub.www.entity;

import jakarta.persistence.Column;
import jakarta.persistence.Entity;
import jakarta.persistence.Id;
import jakarta.persistence.Table;

@Entity
@Table
public class Item 
{
	@Id
	private long itemId;
	
	@Column
	private String itemName;
	
	@Column
	private double itemPrice;

	public long getItemId() {
		return itemId;
	}

	public void setItemId(long itemId) {
		this.itemId = itemId;
	}

	public String getItemName() {
		return itemName;
	}

	public void setItemName(String itemName) {
		this.itemName = itemName;
	}

	public double getItemPrice() {
		return itemPrice;
	}

	public void setItemPrice(double itemPrice) {
		this.itemPrice = itemPrice;
	}
}

ItemController.java

package com.ihub.www.controller;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;

import com.ihub.www.entity.Item;
import com.ihub.www.service.ItemService;

@Controller
public class ItemController 
{
	
	@Autowired
	ItemService itemService;
	
	@RequestMapping("/")
	public String home()
	{
		return "index.jsp";
	}
	
	@RequestMapping("/addItem")
	@ResponseBody
	public String saveItem(Item item)
	{
		return itemService.saveItem(item);
	}
}

package com.ihub.www.service;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import com.ihub.www.entity.Item;
import com.ihub.www.repository.ItemRepository;

@Service
public class ItemService 
{
	@Autowired
	ItemRepository itemRepository;
	
	public String saveItem(Item item)
	{
		itemRepository.save(item);
		return "Item Saved";
	}
}
